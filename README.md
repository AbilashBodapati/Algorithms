# Algorithms
## Sorting Algorithms
### 1. Insertion sort

        Run-Time Complexity: O(n^2)

        Best-Case Run-Time: O(n)

### 2. Merge Sort

        Run-Time Complexity: O(n log(n))

        Best-Case Run-Time: O(n log(n))


### 3. Heap Sort

        Run-Time Complexity: O(n log(n))

        Best-Case Run-Time: O(n log(n))


### 4. Quick Sort

        Run-Time Complexity: O(n^2)

        Best-Case Run-Time: O(n log(n))


### 5. Bubble Sort

        Run-Time Complexity: O(n^2)

        Best-Case Run-Time: O(n+k)


### 6. Counting Sort

        Run-Time Complexity: O(n+k)

        Best-Case Run-Time: O(n+k)


### 7. Radix Sort

        Run-Time Complexity: O(nk)

        Best-Case Run-Time: O(nk)


### 8. Bucket Sort

        Run-Time Complexity: O(n^2)

        Best-Case Run-Time: O(n+k)


### 9. Topological Sort

        Run-Time Complexity: O(V+E)


## Search Algorithms
### 1. Linear Search

        Run-Time Complexity: O(n))


### 2. Binary Search

        Run-Time Complexity: O(log(n)))


### 3. Jump Search 

        Run-Time Complexity: O(Sqrt(n))


### 4. Breadth-First Search

        Run-Time Complexity: O(V+E)

### 5. Depth-First Search 

        Run-Time Complexity: O(V+E)
